# Changelog

All notable changes to **MayaMystic Localization** are documented in this file.

This project follows a structured changelog format to track improvements and new features across releases.

---

## [1.0.0] - 2026-05-26

### ✨ Added
- Initial modular Localization framework architecture
- Centralized `MMLocalization`
- Runtime language switching system
- Runtime language changed event system
- Screen-based localization architecture
- ScriptableObject-driven localization workflow
- `MMLocalizationLanguageSO`
- `MMLoginScreenLocalizationSO`
- Dynamic runtime font switching
- RTL language detection support
- Localization utility architecture
- `MMLocalizationUtility`
- TMP text localization support
- TMP input field localization support
- Runtime UI localization refresh workflows
- Localization alignment override system
- Hebrew localization support
- English localization support
- Runtime localization testing examples
- ContextMenu-based language switching examples
- Keyboard runtime localization testing
- Modular localization package structure
- Reusable localization architecture for production-ready Unity projects

### 🚀 Improved
- Cleaner multilingual UI architecture
- Improved runtime localization workflows
- Better separation of localization responsibilities
- Reusable localization utility pipeline
- Improved UI scalability for multilingual systems
- Better extensibility for future language integrations
- Improved RTL localization workflow support

### 🐛 Fixed
- Fixed runtime localization refresh edge cases
- Fixed missing font application issues
- Fixed RTL alignment inconsistencies
- Fixed language initialization ordering issues
- Fixed TMP input field localization edge cases
